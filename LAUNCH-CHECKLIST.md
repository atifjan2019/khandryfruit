# Launch checklist

All unchecked critical items block production.

- [ ] Domain purchased and HTTPS verified
- [ ] Business email/domain configured; placeholder email removed
- [ ] Address, registration, VAT status and tax details confirmed
- [ ] LUCID and food-business registration confirmed
- [ ] Legal texts approved by qualified German counsel
- [ ] Product list, prices, photos, ingredients, allergens and nutrition received
- [ ] Responsible food business, net weights, storage, batches and BBD confirmed
- [ ] Every public claim has valid verified evidence
- [ ] Shipping carrier/rates and return process confirmed
- [ ] Stripe live keys and signed webhook configured
- [ ] Successful live-mode test order and refund completed
- [ ] Email SPF, DKIM, DMARC, bounce and complaint handling tested
- [ ] Consent categories prevent analytics/marketing before opt-in
- [ ] Mobile Safari, keyboard and WCAG 2.2 AA review completed
- [ ] Sitemap submitted and noindex/canonical/hreflang verified
- [ ] Supabase backups/PITR, restore test and responsible operator confirmed
- [ ] Admin seed password removed and roles reviewed
- [ ] Dependency audit reviewed with no high/critical vulnerabilities
